# tang-vue

[**简体中文**](./README.md) | [**English**](./README.en.md)

#### 项目简介

tang-vue 是基于 Vue3 的权限管理系统

* 采用 Vue3 + Vite4 + Vue Router4 + TypeScript + Pinia + Element Plus
* 自定义多种登录认证方式
* 动态权限菜单
* 代码生成器，一键生成前后端基本代码

#### 环境要求

* Node.js 18.14.0

#### 项目启动

1. ```
   $ yarn && yarn run dev
   ```
2. 访问 [http://localhost:5173/](http://localhost:5173/)
